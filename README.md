# MenuMate
A full-stack web application that helps users discover, share, and interact with recipes based on dietary preferences.

🔗 Live Demo: https://kerolosassad.github.io/MenuMate/

---

## Features
- User-generated recipe creation and management
- Like, comment, and save functionality for user engagement
- API integration for retrieving and displaying recipe data

---

## My Role (Backend-Focused)
- Designed and implemented backend logic to support user interactions (likes, comments, saved recipes)
- Integrated external APIs and handled data processing
- Managed data flow between client and database
- Structured application logic to ensure scalability and maintainability

---

## Tech Stack
- **Backend:** Python
- **Frontend:** JavaScript, HTML, CSS
- **Database:** MongoDB
- **APIs:** RESTful API integration
